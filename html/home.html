About Tortuga 1667

This project is an online recreation of the excellent board game Tortuga 1667 by Facade Games, a game centered around piracy, treason and unreliable alliances. I am not affiliated to Facade Games, and am making this for fun's sake. You can view the source code on GitHub.

Credits

Most of the art for this project was made by Facade Games You can find updated, detailed information about the source of all assets on my GitHub repository.

Help

This is the home page of the game, where you can set several settings for the game. Once you have selected your settings, clicking Play will create a new game, where all your friends can join. When everyone is there, you can start the game!

  • Event hints: Shows a small icon next to the event cards you have already viewed.
  • All players: Shows on each event card the icons of the players who have viewed it.
  • Card reveal: Leave the event cards revealed (face up) once you have viewed them.
